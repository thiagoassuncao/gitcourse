********COMANDOS GIT********

=> 1 - Configuração Inicial <=

1.1 - Configuração de Username

$git config --global user.name "Nome"

1.2 - Configuração de e-mail

$git config --global user.email "email@mail.com"

1.3 - Configuração de editor

$git config --global core.editor nomeEditor(Padrão do Linux: vim)

1.4 - Exibir valores de já configurados

$git config user.name (Exibe nome do usuario)

$git config user.email (Exibe email do usuario)

$git config --list (Exibe todas as configuraçõe)



=> 2 - COMANDOS ESSENCIAIS <=

2.1 Inicializando projeto

$mkdir nomeDaPasta (Cria a pasta do projeto)

$cd nomeDaPasta (Entra na pasta)

$git init (Inicia o repositorio)

2.2 - Estatus do Git

$git status

2.3 - Adicionar arquivo no repositorio

$git add nomeDoArquivo

2.4 - Salvar arquivo no repositorio

$git commit -m "Comentario sobre modificação"
